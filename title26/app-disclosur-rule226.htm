Rule 226. Joinder of Parties

The joinder of parties in a disclosure action shall be subject to the following requirements:

(a) Commencement of Action: Any person who meets the requirements for commencing such an action may join with any other such person in filing a petition with respect to the same written determination, prior written determination, or background file document. But see Code Sections 6110(f)(3)(B) and (h)(4).

(b) Consolidation of Actions: If more than one petition is filed with respect to the same written determination, prior written determination, or background file document, then see Rule 141 with respect to the consolidation of the actions.
